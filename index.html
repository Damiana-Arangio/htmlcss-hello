Hello Git – Una giornata al museo

Pagina HTML statica che riproduce un semplice layout “Una giornata al museo” e mi ha permesso di prendere confidenza con Git e con la struttura base di una pagina HTML. ## Obiettivo - Creare la cartella del progetto e aprirla con VS Code - Creare un file HTML di base - Inizializzare il repository Git ed effettuare il primo commit - Effettuare il primo push su GitHub - Completare progressivamente il layout: - intestazione con titolo della pagina - sezione con le opere d'arte (immagini e descrizioni) - sezione con elenco puntato delle cose imparate - piè di pagina con copyright ## Anteprima ![Anteprima header-main](img/ScreenshotTitoloDipintoScultura.png) ![Anteprima main-footer](img/ScreenshotApprendimento+Footer.png) ## Tecnologie utilizzate - HTML5 - Git
